
1：usearch -fastq_mergepairs *_R1.fq -fastqout merged.fq  -relabel @ -fastq_maxdiffs 0  
# Match all forward sequences *_R1.fq to automatically find the corresponding R2.fq.

2：usearch -fastq_filter merged.fq -fastq_trunclen 300 -fastq_maxee 0.5 -fastaout filtered.fa
#-fastq_trunclen 300 (The value here is to set the length according to your primer)

3：usearch -fastx_uniques filtered.fa -fastaout uniques.fa -sizeout -relabel Uniq
#Find unique sequences and count the number.

4：usearch -cluster_otus uniques.fa -otus otus.fa -uparseout out.up -relabel OTU -minsize 2
#OTUs cluster （Clustering defaults to 97%, but it is possible to set it to 95% or 99%）

5:usearch -otutab merged.fq -otus otus.fa -otutabout otutab_raw.txt
#Generate OTU table and represent sequences.
